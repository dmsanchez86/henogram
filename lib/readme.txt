This package contains both library and examples of using FireShot API:

"fsapi.js"  - a script providing connectivity between user web page and FireShot functions.
"demo1.htm" - an example of how to detect whether FireShot is available or not.
"demo2.htm" - an example of how to launch different FireShot's functions.
"demo3.htm" - live FireShot application.

For additional info please refer to http://screenshot-program.com/fireshot/api_demo.php
and http://code.google.com/p/fireshot-api/
